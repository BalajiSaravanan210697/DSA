# DSA

Repo for problem solving using Data Structures and Algorithms in JavaScript.

## Structure

| Folder | Topics |
|--------|--------|
| `arrays` | Two pointers, sliding window, prefix sums |
| `strings` | Pattern matching, anagrams, palindromes |
| `linked-lists` | Reversal, cycle detection, merge |
| `stacks-and-queues` | Monotonic stack, BFS, deque |
| `trees` | BST, traversals, LCA |
| `graphs` | DFS, BFS, shortest path, topological sort |
| `dynamic-programming` | Memoization, tabulation, common patterns |
| `sorting-and-searching` | Binary search, merge sort, quick sort |
| `recursion` | Backtracking, divide and conquer |
| `hash-tables` | Frequency maps, two-sum patterns |

## File naming

Each solution file: `<problem-name>.js` (e.g., `two-sum.js`)
